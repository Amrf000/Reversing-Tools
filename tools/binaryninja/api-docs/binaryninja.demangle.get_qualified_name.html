binaryninja.demangle.get_qualified_name¶

binaryninja.demangle.get_qualified_name(names)[source]¶

get_qualified_name gets a qualified name for the provied name list.

Parameters:

names (list(str)) – name list to qualify
Returns:

a qualified name
Return type:

str
Example:
>>> type, name = demangle_ms(Architecture["x86_64"], "?testf@Foobar@@SA?AW4foo@1@W421@@Z")
>>> get_qualified_name(name)
'Foobar::testf'
>>>
