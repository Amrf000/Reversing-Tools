binaryninja.log.log_alert¶

binaryninja.log.log_alert(text)[source]¶

log_alert Logs message console and to a pop up window if run through the GUI.

Parameters:

text (str) – message to print
Return type:

None
Example:
>>> log_to_stdout(LogLevel.DebugLog)
>>> log_alert("Kielbasa!")
Kielbasa!
>>>
